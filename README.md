# Plataforma de Apoio Emocional com IA

`CURSO`

`DISCIPLINA`

`SEMESTRE`

Plataforma web de apoio emocional que permite o registro diário de humor e desabafos em texto livre. A aplicação utiliza análise de sentimento com Inteligência Artificial para identificar padrões emocionais ao longo do tempo, oferece dashboard com gráficos semanais e mensais, detecção de palavras de risco com exibição de contatos de ajuda (como o CVV – 188) e modo "acalmar agora" com técnicas de respiração e relaxamento.

**Importante:** o sistema não substitui psicólogos ou terapia e não realiza diagnóstico psicológico; é uma ferramenta de apoio e autoconsciência emocional.

**ODS alinhados:** ODS 3 – Saúde e Bem-Estar (garantir uma vida saudável e promover o bem-estar para todos).
